### PdfWriter

- Responsável por escrever conteúdo no documento.

- Utilizado quando você deseja criar um novo documento PDF ou sobrescrever um existente.

## Parâmetros aceitos

- *MemoryStream*: Para criar um documento PDF em memória

### PdfDocument

- Deve ser criado passando no construtor ou o PdfWriter para criar um novo documento ou sobreescrer um novo PDF ou o PdfReader para ler algum PDF existente.

## Parâmetros aceitos

- *PdfWriter*
- *PdfReader*
